# 实验指导书功能集成到考核生成页面

## 📋 集成步骤

### 1. 前端集成

#### 步骤1：在 ExamGeneration.vue 中引入组件

在 `<script setup>` 或 `<script>` 部分的 import 区域添加：

```javascript
import ExperimentGuideDialog from './components/ExperimentGuideDialog.vue'
```

#### 步骤2：添加响应式变量

在 setup() 函数或 data() 中添加：

```javascript
const showExperimentGuideDialog = ref(false)
```

#### 步骤3：在 template 末尾添加组件

在 `</template>` 标签之前添加：

```vue
<!-- 实验指导书生成对话框 -->
<ExperimentGuideDialog 
  v-model="showExperimentGuideDialog"
  @success="handleExperimentGuideSuccess"
/>
```

#### 步骤4：添加成功回调方法

```javascript
const handleExperimentGuideSuccess = () => {
  ElMessage.success('实验指导书生成成功')
}
```

#### 步骤5：按钮已添加

按钮已经添加在考核设置卡片中：
```vue
<!-- 实验指导书生成按钮 -->
<div style="margin-top: 12px;">
  <el-button @click="showExperimentGuideDialog = true" icon="Document" type="success" style="width: 100%;">
    📝 生成实验指导书
  </el-button>
</div>
```

### 3. API接口已完成

前端API接口已在 `src/api/smartTeaching.js` 中定义：

```javascript
// 生成实验指导书
export const generateExperimentGuide = (data) => {
  return request({
    url: '/api/smart-teaching/generate-experiment-guide',
    method: 'post',
    data
  })
}

// 获取实验指导书内容
export const getExperimentGuide = (guideId) => {
  return request({
    url: `/api/smart-teaching/experiment-guide/${guideId}`,
    method: 'get'
  })
}

// 获取实验指导书列表
export const getExperimentGuides = (teacherId) => {
  return request({
    url: `/api/smart-teaching/experiment-guides/${teacherId}`,
    method: 'get'
  })
}

// 删除实验指导书
export const deleteExperimentGuide = (guideId) => {
  return request({
    url: `/api/smart-teaching/experiment-guide/${guideId}`,

  })
```

```vue
// ... 其他变量定义

// 实验指导书对话框
const showExperimentGuideDialog = ref(false)

const handleExperimentGuideSuccess = () => {
  ElMessage.success('实验指导书生成成功')
}

// ... 其他代码
</script>

<template>
  <!-- ... 现有内容 -->
  
  <!-- 在 template 末尾，</template> 之前添加 -->
  <ExperimentGuideDialog 
    v-model="showExperimentGuideDialog"
    @success="handleExperimentGuideSuccess"
  />
</template>
```

```vue

      this.$message.success('实验指导书生成成功')
```

## 🎯 功能特点

1. **AI驱动生成**：使用通义千问AI自动生成实验指导书
2. **完整结构**：包含实验目的、环境、步骤、代码、结果、思考题、答案
3. **异步处理**：后端异步生成，前端轮询检查状态
4. **历史记录**：支持查看历史生成记录
5. **详情查看**：格式化展示实验指导书各部分内容
6. **删除管理**：支持删除不需要的记录
