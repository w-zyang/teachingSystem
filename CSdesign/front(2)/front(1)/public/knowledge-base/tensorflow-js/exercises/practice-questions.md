# TensorFlow.js 练习评估题目库

## 基础概念类题目

### 1. TensorFlow.js的主要优势不包括？
- [ ] A. 支持浏览器端GPU加速
- [ ] B. 可以与Python版TensorFlow模型互转
- [ ] C. 使用JavaScript语言开发
- [x] D. 只能在服务器端运行

**答案**: D
**解析**: TensorFlow.js可以在浏览器和Node.js环境中运行，不仅限于服务器端。

### 2. 在TensorFlow.js中，张量（Tensor）的特点是什么？
- [x] A. 是不可变的数据结构
- [ ] B. 值可以随时修改
- [ ] C. 只能是一维数组
- [ ] D. 不支持数学运算

**答案**: A
**解析**: 张量是不可变的数据结构，操作会返回新的张量，不会修改原始值。

### 3. TensorFlow.js中，tf.tidy()函数的主要作用是？
- [ ] A. 提高计算速度
- [x] B. 自动管理GPU内存，防止内存泄漏
- [ ] C. 优化模型结构
- [ ] D. 压缩模型大小

**答案**: B
**解析**: tf.tidy()函数用于自动清理中间张量，释放GPU内存，防止内存泄漏。

### 4. 在TensorFlow.js中，创建Sequential模型的方法是？
- [ ] A. tf.model()
- [x] B. tf.sequential()
- [ ] C. tf.layers.sequential()
- [ ] D. tf.createModel()

**答案**: B
**解析**: tf.sequential()是创建Sequential模型的正确方法。

### 5. TensorFlow.js支持的后端不包括？
- [ ] A. CPU后端
- [ ] B. WebGL后端
- [ ] C. WASM后端
- [x] D. Python后端

**答案**: D
**解析**: TensorFlow.js是JavaScript版本，不支持Python后端，但可以与Python模型互转。
