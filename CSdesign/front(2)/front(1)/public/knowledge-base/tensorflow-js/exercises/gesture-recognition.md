# 手势识别项目单选题

## 项目概述与目标

### 1. "剪刀石头布"手势识别项目属于什么类型的机器学习任务？
- [ ] A. 回归任务
- [x] B. 图像分类任务
- [ ] C. 目标检测任务
- [ ] D. 语义分割任务

**答案**: B
**解析**: 手势识别是将输入的手势图像分类为剪刀、石头、布三种类别，属于图像分类任务。

### 2. 手势识别项目的主要目标是？
- [ ] A. 识别手势的精确位置
- [x] B. 根据摄像头采集的手势图像确定代表剪刀、石头、布中的哪一个
- [ ] C. 跟踪手势的运动轨迹
- [ ] D. 生成手势的3D模型

**答案**: B
**解析**: 项目目标是使用TensorFlow.js实现根据摄像头采集的手势图像来确定它代表剪刀、石头、布中的哪一个。

### 3. 手势识别项目的工作流程不包括？
- [ ] A. 数据图像采集
- [ ] B. 模型训练
- [ ] C. 参数调整
- [x] D. 语音识别

**答案**: D
**解析**: 手势识别项目主要涉及图像处理，不包括语音识别功能。

## 模型架构与设计

### 7. 手势识别项目最适合使用的模型架构是？
- [ ] A. 全连接神经网络
- [x] B. 卷积神经网络（CNN）
- [ ] C. 循环神经网络（RNN）
- [ ] D. 自编码器

**答案**: B
**解析**: 手势识别是图像分类任务，卷积神经网络（CNN）最适合处理图像数据，能够有效提取空间特征。

### 8. 在CNN模型中，卷积层的主要作用是？
- [ ] A. 减少模型参数
- [x] B. 提取图像的空间特征
- [ ] C. 降低计算复杂度
- [ ] D. 增加模型深度

**答案**: B
**解析**: 卷积层通过滑动卷积核来提取图像的空间特征，如边缘、纹理等。

### 9. 对于手势识别任务，输入图像的典型尺寸可能是？
- [ ] A. 28x28像素
- [ ] B. 64x64像素
- [x] C. 224x224像素
- [ ] D. 512x512像素

**答案**: C
**解析**: 224x224像素是常见的图像分类输入尺寸，适合手势识别任务。

### 10. 手势识别模型的输出层应该有多少个神经元？
- [ ] A. 1个
- [ ] B. 2个
- [x] C. 3个

**答案**: C
**解析**: 手势识别需要分类为剪刀、石头、布三个类别，因此输出层应该有3个神经元。
