# TensorFlow.js 项目实践单选题

## 汽车油耗预测项目

### 1. 在汽车油耗预测项目中，数据预处理的主要步骤不包括以下哪项？
- [ ] A. 数据清洗和过滤
- [ ] B. 数据归一化处理
- [x] C. 图像尺寸调整
- [ ] D. 张量格式转换

**答案**: C
**解析**: 汽车油耗预测是回归问题，处理的是数值数据，不需要图像尺寸调整。图像尺寸调整主要用于图像分类任务。

### 2. 汽车油耗预测项目中使用的模型结构是？
- [ ] A. 卷积神经网络（CNN）
- [x] B. 全连接神经网络
- [ ] C. 循环神经网络（RNN）
- [ ] D. 生成对抗网络（GAN）

**答案**: B
**解析**: 汽车油耗预测是简单的线性回归问题，使用全连接神经网络即可，输入是功率，输出是油耗。

### 3. 在汽车油耗预测项目中，tf.tidy()函数的主要作用是？
- [ ] A. 提高模型训练速度
- [x] B. 自动管理GPU内存，防止内存泄漏
- [ ] C. 优化模型结构
- [ ] D. 提高预测准确率

**答案**: B
**解析**: tf.tidy()函数用于自动清理中间张量，释放GPU内存，防止内存泄漏，这是TensorFlow.js中重要的内存管理机制。

### 4. 汽车油耗预测项目中，数据归一化的目的是？
- [ ] A. 减少数据量
- [x] B. 将数据映射到0-1范围，提高训练效率
- [ ] C. 增加数据复杂度
- [ ] D. 改变数据分布

**答案**: B
**解析**: 归一化将数据映射到0-1范围，使不同尺度的特征具有可比性，提高模型训练的稳定性和收敛速度。

### 5. 在汽车油耗预测项目中，模型训练时使用的损失函数是？
- [ ] A. 交叉熵损失
- [x] B. 均方误差（MSE）
- [ ] C. 绝对误差
- [ ] D. 对数损失

**答案**: B
**解析**: 汽车油耗预测是回归问题，使用均方误差（MSE）作为损失函数，用于衡量预测值与真实值之间的差异。

## 手写数字识别项目

### 6. MNIST手写数字识别项目中，输入图像的尺寸是？
- [ ] A. 32x32像素
- [x] B. 28x28像素
- [ ] C. 64x64像素
- [ ] D. 16x16像素

**答案**: B
**解析**: MNIST数据集中的手写数字图像是28x28像素的灰度图像，这是该数据集的标准尺寸。

### 7. 手写数字识别项目中使用的模型类型是？
- [x] A. 卷积神经网络（CNN）
- [ ] B. 全连接神经网络
- [ ] C. 循环神经网络（RNN）
- [ ] D. 自编码器

**答案**: A
**解析**: 手写数字识别是图像分类任务，使用卷积神经网络（CNN）能够有效提取图像的空间特征，提高识别准确率。

### 8. 在CNN模型中，MaxPooling层的主要作用是？
- [ ] A. 增加模型参数
- [x] B. 降维和提取主要特征
- [ ] C. 增加模型复杂度
- [ ] D. 提高训练速度

**答案**: B
**解析**: MaxPooling层通过取滑动窗口内的最大值来降维，同时保留主要特征信息，减少计算量并提高模型的泛化能力。

### 9. 手写数字识别项目中，输出层的激活函数是？
- [ ] A. ReLU
- [ ] B. Sigmoid
- [x] C. Softmax
- [ ] D. Tanh

**答案**: C
**解析**: 手写数字识别是多分类问题（0-9共10个类别），使用Softmax激活函数将输出转换为概率分布，所有概率和为1。

### 10. 在模型评估中，混淆矩阵的作用是？
- [ ] A. 显示模型训练过程
- [x] B. 可视化分类结果，显示各类别的预测准确性
- [ ] C. 优化模型参数
- [ ] D. 提高训练速度

**答案**: B
**解析**: 混淆矩阵是评估分类模型性能的重要工具，可以清楚地显示每个类别的预测准确性，帮助识别模型的优势和不足。

## 项目实践要点

### 11. 在TensorFlow.js项目中，tfvis的主要作用是？
- [ ] A. 模型训练
- [x] B. 数据可视化和训练过程监控
- [ ] C. 数据预处理
- [ ] D. 模型部署

**答案**: B
**解析**: tfvis是TensorFlow.js的可视化工具库，用于在浏览器中显示数据分布、训练过程和模型性能等可视化信息。

### 12. 在浏览器端运行机器学习模型的优势不包括？
- [ ] A. 减少服务器计算压力
- [ ] B. 提高响应速度
- [ ] C. 保护用户隐私
- [x] D. 计算能力更强

**答案**: D
**解析**: 浏览器端的计算能力通常不如服务器端，但具有减少服务器压力、提高响应速度和保护隐私等优势。

### 13. TensorFlow.js中，模型训练时的batchSize设置考虑因素不包括？
- [ ] A. 内存限制
- [ ] B. 训练速度
- [x] C. 网络连接速度
- [ ] D. 模型收敛性

**答案**: C
**解析**: batchSize主要考虑内存限制、训练速度和模型收敛性，网络连接速度不是主要考虑因素。

### 14. 在项目开发中，使用构建工具（如Parcel）的优势是？
- [ ] A. 简化代码编写
- [x] B. 支持模块化开发和代码优化
- [ ] C. 提高模型准确率
- [ ] D. 减少数据量

**答案**: B
**解析**: 构建工具支持模块化开发、代码压缩优化、依赖管理等功能，提高开发效率和代码质量。

### 15. TensorFlow.js模型部署时，需要考虑的主要因素是？
- [ ] A. 模型文件大小和加载速度
- [ ] B. 浏览器兼容性
- [ ] C. 硬件加速支持
- [x] D. 以上都是

**答案**: D
**解析**: TensorFlow.js模型部署需要考虑模型文件大小、加载速度、浏览器兼容性和硬件加速支持等多个因素。
