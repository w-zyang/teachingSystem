# 本地知识库资料

本目录包含用于AI大模型本地知识库的专业相关资料内容。

## 目录结构

```
knowledge-base/
├── computer-science/          # 计算机科学相关
│   ├── programming/          # 编程语言
│   ├── algorithms/           # 算法与数据结构
│   ├── database/             # 数据库
│   ├── web-development/      # Web开发
│   └── software-engineering/ # 软件工程
├── mathematics/              # 数学相关
│   ├── calculus/            # 微积分
│   ├── linear-algebra/      # 线性代数
│   └── statistics/          # 统计学
├── physics/                  # 物理学
├── chemistry/                # 化学
├── biology/                  # 生物学
└── general/                  # 通用知识
```

## 使用说明

1. 所有文档采用Markdown格式
2. 文档内容应包含完整的知识点和示例
3. 定期更新知识库内容以保持准确性
4. 支持多语言内容（中文为主）

## 文件命名规范

- 使用小写字母和连字符
- 文件名应具有描述性
- 按学科和主题分类组织
