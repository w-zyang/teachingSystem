## ⚠️ 重要注意事项

### 1. 数据库备份
在导入演示数据前，请务必备份现有数据：
```bash
mysqldump -u root -p web_experiment > backup_$(date +%Y%m%d).sql
```

### 2. 目录重命名
由于 `front(2)/front(1)` 目录正在被使用，需要：
1. 关闭所有运行中的服务（前端、后端）
2. 关闭IDE（VS Code/Cursor）
3. 按照OPTIMIZATION_REPORT.md中的步骤手动重命名

### 3. 配置文件更新
如果重命名了目录，需要更新：
- README.md 中的路径引用
- 启动脚本中的路径
- 其他文档中的路径说明

### 4. 浏览器缓存
测试新UI时，建议：
- 清除浏览器缓存
- 使用无痕模式
- 强制刷新（Ctrl+F5）
