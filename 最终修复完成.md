## 📝 Vue3 Ref 使用规则总结

### 在 script 中
```javascript
const examConfig = ref({ type: 'exam', name: '', duration: 60 })

// 访问和修改都需要 .value
console.log(examConfig.value.type)
examConfig.value.type = 'experiment'
examConfig.value.name = '测试'
```

### 在 template 中
```vue
<!-- 访问ref对象的属性时，必须使用 .value -->
<input v-model="examConfig.value.name">
<div v-if="examConfig.value.type === 'exam'">
  {{ examConfig.value.name }}
</div>
```
